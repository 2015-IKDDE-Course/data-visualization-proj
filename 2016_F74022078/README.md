#動機
人們在旅遊的時候通常會希望能夠知道哪邊的景點最多，這樣才能在相同時間內拜訪最多地方，出於此發想，本程式統計台南各區的景點數量，並以柱狀圖的方式向使用者呈現台南各區的景點數量
#open data 來源
http://data.gov.tw/node/gov/resource/309
