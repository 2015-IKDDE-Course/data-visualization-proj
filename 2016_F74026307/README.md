題目:Top 25 Students Number in Tainan Junior High School

作者:王郁強(Yu-Chiang Wang)

作品目的:
	以網頁資料視覺化最常見的d3.js來做為學習Data Visualization的工具
	各學校在學人數是學校辦學成果的重要參考依據，也是評斷台南市區域發展的參考指標
	故以此來做為視覺化的來源資料

使用技術:
	d3.v4.min.js
	jquery.min.js
	
設計流程:
	1.尋找可用資料
	2.分析資料、觀察資料特性
	3.選擇資料視覺化的圖表類型
	5.實作

資料來源:
	政府資料開放平台-臺南市國中各校男女生人數統計
	http://data.gov.tw/node/6181
	
測試環境:
	Mozilla Firefox
